Paradise Nursery is a simple shopping cart web application built using React and Redux.  
The application allows users to browse a variety of houseplants, add them to a shopping cart, and manage quantities dynamically.

## Features
- Landing page with company introduction
- Product listing page with categorized plants
- Shopping cart with quantity control and total cost
- Redux-based state management
- Deployed using GitHub Pages

## Technologies Used
- React
- Redux Toolkit
- JavaScript (ES6)
- CSS
- GitHub Pages
